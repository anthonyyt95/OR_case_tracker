⚙ Google Sheets Setup

ONE-TIME CONFIGURATION · TAKES ~10 MINUTES

Follow these steps once. After that, every "Save to Sheets" sends your case data directly to your Google Sheet — searchable and filterable from the Sheets app on your iPhone.

  1. Go to sheets.google.com and create a new blank spreadsheet. Name it something like OR Case Log.
  2. In that spreadsheet, click Extensions → Apps Script. A code editor will open.
  3. Delete all existing code in the editor, then paste in the Apps Script code from the separate apps_script.gs file provided.
  4. Click Save (disk icon), then click Deploy → New deployment.
  5. Click the gear icon → choose Web app. Set "Execute as" = Me, "Who has access" = Anyone. Click Deploy.
  6. Google will ask you to authorize — click Authorize access, sign in, and click Advanced → Go to OR Case Log (unsafe) if prompted.
  7. Copy the Web app URL (looks like https://script.google.com/macros/s/ABC.../exec) and paste it below.

Your URL is saved locally in this browser. It never leaves your device except when sending case data to your own Google Sheet.
